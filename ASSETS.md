# Image Asset Requirements

## Favicon
- File format: ICO or PNG
- Dimensions: 16x16 px or 32x32 px
- Should be optimized for small icon display.

## Apple Touch Icon
- File format: PNG
- Dimensions: 180x180 px
- Should be used for adding to home screen on iOS devices.

## Preview.png
- File format: PNG
- Dimensions: 1200x630 px
- Recommended for social media sharing previews and should have a suitable aspect ratio (1.91:1).
